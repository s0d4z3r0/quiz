# **Projeto Quiz**
## _Quiz em React_

Projeto **Quiz** feito em **React + Vite**.
O projeto consiste em um quiz de perguntas que armazenam as respostas do usuário e ao final é mostrado um gabarito. Estilizado em SASS e consumindo uma fakeAPI.
